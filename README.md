# Rental System by Hamdi

نظام إدارة مستأجرين شامل يدعم العمليات التالية:

- تسجيل مستأجر جديد  
- تعديل بيانات مستأجر  
- حذف مستأجر  
- الاستعلام عن المستأجرين  
- إدارة الشؤون المالية  

الواجهة: HTML / CSS  
الخلفية: PHP + MySQL  
المشروع مستضاف على: [hamdiimage.infy.uk](https://hamdiimage.infy.uk/FilePHP2/)
